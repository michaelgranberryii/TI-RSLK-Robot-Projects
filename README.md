# TI RSLK Robot Projects
A collection of robotics projects developed on the TI-RSLK robot kit. The project span a broad range of topics from serial communication, tachometer, light sensors, distance sensors, timers, ACDs, GPIOs, and object avoidance / follower algorithms.

## Projects
* [ADC - Analog IR Distance Sensor - Object Avoidance / Follower](https://github.com/michaelgranberryii/ti-rslk-adc-object-avoidance-follower)
* [I2C Light and IR Distance Sensor](https://github.com/michaelgranberryii/ti-rslk-i2c-light-distance-sensor)
* [Line Follower Robot](https://github.com/michaelgranberryii/ti-rslk-line-follower)
* [UART - SPI](https://github.com/michaelgranberryii/ti-rslk-uart-spi)
* [Tachometer](https://github.com/michaelgranberryii/ti-rslk-tachometer)
* [PWM for Servo Motor and Gearmotor](https://github.com/michaelgranberryii/ti-rslk-pwm-servo-motor)
* [Timers - Interrupts](https://github.com/michaelgranberryii/ti-rslk-timer-interrupts)
* [GPIO - LEDs - Switches - Push Buttons](https://github.com/michaelgranberryii/ti-rslk-gpio)
